

Provides a good starting point for an Android project:
* Kotlin
* Dagger2
* RxJava2
* OkHttp3
* Retrofit2
* Glide4
* Architecture Components (LiveData, ViewModel, Room)
* Stetho (present only in debug builds)
* Travis CI integration; every commit pushed to github is built
